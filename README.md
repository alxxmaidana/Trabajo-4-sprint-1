# Trabajo Práctico 4 - Sprint 1: Express y Arquitectura MVC

Repositorio con ejercicios progresivos sobre **Node.js**, **Express.js** y la **arquitectura Modelo-Vista-Controlador (MVC)**.

## 📁 Estructura del Repositorio

### 1. **actividad-1** - Manejo de Parámetros de Ruta
- Servidor Express básico que recibe parámetros en la URL (`:id`)
- Muestra el parámetro en consola
- **Objetivo:** Entender cómo funciona el enrutamiento dinámico

### 2. **actividad-2** - Manejo de Parámetros de Consulta
- Servidor Express que recibe parámetros de consulta (`?edad=25`)
- Procesa y muestra los parámetros en consola
- **Objetivo:** Diferenciar entre parámetros de ruta y parámetros de consulta

### 3. **ejemplo-mvc** - Ejemplo de Arquitectura MVC
- Ejemplo de Servidor con patrón MVC completo para gestión de **tareas**
- Estructura con controladores, modelos, servicios, repositorio y vistas
- **Objetivo:** Comprender la separación de responsabilidades en MVC

### 4. **proyect-root** - API REST de Superhéroes 
- API REST completa con **arquitectura MVC**
- Endpoints para obtener superhéroes por ID, atributo y edad
- Incluye capas de:
  - **Controllers** (lógica de solicitud)
  - **Services** (lógica de negocio)
  - **Repository** (acceso a datos)
  - **Views** (formateo de respuestas)
- **Objetivo:** Integrar todos los conceptos aprendidos
